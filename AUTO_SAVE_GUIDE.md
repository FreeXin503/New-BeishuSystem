#  自动保存到 GitHub 使用指南

##  可用的 npm 脚本

### 1. **自动保存并推送** (推荐)
```bash
npm run save
```

### 2. **快速保存**
```bash
npm run quick-save
```

### 3. **本地保存** (不推送)
```bash
npm run save-no-push
```

##  VS Code 快捷键

- **Ctrl + S**  自动保存到 Git 并推送到 GitHub
- **Ctrl + Shift + S**  快速保存到 Git 并推送到 GitHub

##  现在你可以：

1. 编辑文件后按 `Ctrl + S`
2. 自动提交并推送到 GitHub
3. 再也不用担心代码丢失！
